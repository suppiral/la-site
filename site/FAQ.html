In which programming language was Little Adventure made?

The programming language used to develop Little Adventure is C++ using Simple Fast Media Library (SFML) for graphics.

Why did you make Little Adventure?

Little Adventure is my final project at OOP2 class of 2013 in Hadassah college. I choose to do this kind of a game because this is my favorite genre and I always wanted to make games so this was a good opportunity for me to try and create a game on my own.

How long did it took to make Little Adventure?

Creating Little Adventure took a about a month to make.

Why is Little Adventure so short?

Since only one person worked on this project most of the work focused on creating a good engine that can create a variety of game using simple scripts that will describe the content of the game. therefore little time was left for making the game plot and content.

I can't hear any sounds. why is that?

Unfortunately Little Adventure has no sounds. there simply wasn't enough time to work on that.

How do I move around the game?

The controls for the game are:

Arrows
for Movement
Z button
for Action/Attack
X button
to toggle run
A and S buttons
to switch between attack types
ESC
to go back or pop in game menu

Where is the save button?

You cannot save during this game. but it's ok because its a short game.
